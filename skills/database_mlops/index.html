Databases & MLOps

Experienced in working with databases like SQLite, MongoDB, and Apache Cassandra, along with MLOps tools such as GIT, DVC (Data Version Control), MLflow, CircleCI, and Docker, ensuring seamless data management, version control, and deployment in machine learning projects.

Databases:

  • SQLite: A lightweight, serverless, and self-contained SQL database engine.
  • MongoDB: A flexible, scalable, and document-oriented NoSQL database.
  • Apache Cassandra: A highly scalable and distributed NoSQL database known for its fault-tolerance and high availability.

MLOps:

  • GIT: A widely used distributed version control system for tracking changes in code and collaborating with others.
  • DVC (Data Version Control): An open-source version control system specifically designed for handling machine learning projects and their datasets.
  • MLflow: An open-source platform for managing the machine learning lifecycle, including experimentation, reproducibility, and deployment.
  • CircleCI: A continuous integration and delivery platform that automates the build, test, and deployment processes.
  • Docker: A containerization platform that simplifies the packaging and deployment of applications in isolated and reproducible environments.
