Fingerprint Verification System

July 3, 2022

PROJECT REPORT: FINGERPRINT VERIFICATION SYSTEM

Led a team of six data scientists and engineers to design and implement a highly secure biometric user authentication system using Neural Fuzzy Extractor (NFE) and Artificial Neural Networks (ANN), achieving a marginal improvement in the equal error rate (EER) from 5.75% to 5.82%, while ensuring minimal loss of FVC2006 fingerprint data. Enhanced the precision of the fingerprint data by formulating a robust deep learning algorithm, specifically a Siamese Network with a triplet loss function, resulting in an Area Under Curve (AUC) score of 0.986 using TensorFlow and Keras.

INTRODUCTION

The Fingerprint Verification System project aims to develop a highly secure biometric user authentication system based on fingerprint recognition. The objective is to design an accurate and reliable system that can verify a user’s identity by comparing their fingerprint with the stored reference fingerprint. The project incorporates Neural Fuzzy Extractor (NFE) and Artificial Neural Networks (ANN) to enhance the accuracy and efficiency of fingerprint verification.

BIOMETRIC AUTHENTICATION

Biometric authentication is a method of verifying a user’s identity based on their unique physical features, such as fingerprints, face, retina scan, etc. Unlike identification, which aims to identify a person, authentication focuses on verifying if a user is who they claim to be. Biometric authentication provides an additional layer of security on top of traditional password-based authentication.

ONE-SHOT CLASSIFICATION AND SIAMESE NETWORK

In this project, we applied the concept of one-shot classification and employ a Siamese network to create a fingerprint authentication system. One-shot classification treats the task as a “difference evaluation problem” rather than a classification problem. The Siamese network takes two input fingerprint images, processes them through the same network, and generates vector embeddings. These embeddings are then used to calculate a similarity score between the two images, enabling fingerprint verification.

SIAMESE NETWORK ARCHITECTURE

The Siamese network architecture consists of a triplet loss layer and a modified VGG16 model. The triplet loss layer is responsible for generating embeddings and calculating the similarity score between the anchor image (reference fingerprint), positive image (same user’s fingerprint), and negative image (different user’s fingerprint). The VGG16 model, pre-trained on ImageNet, is used as the base network for feature extraction.

TRIPLET LOSS FUNCTION

The triplet loss function plays a crucial role in the Siamese network architecture. It learns to maximize the similarity between similar images (anchor-positive pair) and minimize the distance between dissimilar images (anchor-negative pair). By using the triplet loss function, the model extracts more features and generates more training samples, leading to improved accuracy.

SET-UP AND TRAINING

The FVC2006 dataset, consisting of four distinct subsets (DB1, DB2, DB3, and DB4), is used for training and testing. For training, 10 impressions per finger from DB1-A are used to generate “hard” and “easy” triplet pairs. The Siamese network is trained using TensorFlow and Keras. For testing, two impressions per finger from DB1-A are used.

RESULTS AND EVALUATION

The trained model achieved an accuracy of 95.36% on the testing data, with only 13 mismatches out of 280 fingerprint impressions. The system’s decoding region radius of 0.7 ensured reliable verification for all users. The Receiver Operating Characteristic (ROC) curve showed an impressive Area Under Curve (AUC) score of 0.986, indicating a high level of performance.

CONCLUSION

The Fingerprint Verification System project successfully designed and implemented a secure biometric user authentication system using Neural Fuzzy Extractor (NFE) and Artificial Neural Networks (ANN). The system leverages a Siamese network with a triplet loss function to accurately verify a user’s identity based on their fingerprint. The system achieved a marginal improvement in the equal error rate (EER) and demonstrated an impressive AUC score of 0.986.
